# 🎵 MP3 Tag Filler

Автоматическое заполнение ID3-тегов MP3 файлов через распознавание Shazam.

## Установка

```bash
pip install MelodyMaster mutagen
```

**Требование:** Python 3.8+

## Использование

```bash
python tag_filler.py <файл.mp3> [опции]
```

### Опции

| Опция | Тег | Описание |
|-------|-----|----------|
| `-t` | TIT2 | Название трека |
| `-a` | TPE1 | Исполнитель |
| `-m` | TALB | Альбом |
| `-y` | TDRC | Год выпуска |
| `-g` | TCON | Жанр |
| `-l` | TLAN | Язык |

### Пример

```bash
# Заполнить название и исполнителя
python tag_filler.py song.mp3 -t -a

# Заполнить все доступные теги
python tag_filler.py song.mp3 -t -a -m -y -g -l
```

## Как это работает

1. Распознаёт трек через Shazam
2. Ищет язык трека в MusicBrainz
3. Записывает полученные данные в ID3-теги файла

## Примечание

Скрипт обрабатывает **только MP3 файлы**. При отсутствии распознавания теги не будут изменены.
